# RxJS-Lunch-n-Learn
Slides and some code for my Lunch 'n' Learn.
Took place on 2019-06-11.

## Example code

Edit code in the `src/` folder.

To run:
```shell
npm start
```

Open http://localhost:8080/

## Slides

To run:
```shell
npm run slides
```

Open http://localhost:6275/
